# 한국 유튜브 뮤직 차트 추출

## 기능
한국 유튜브 뮤직 차트의 제목, 채널, 길이, 조회수, 링크를 추출하여 `{오늘}_database.csv` 파일로 저장합니다.

## 실행 방법

1. 필요한 패키지를 설치합니다:
    ```bash
    pip install pytubefix csv datetime
    ```

2. 스크립트를 실행합니다:
    ```bash
    python index.py
    ```

3. (옵션) 유튜브 프리미엄 사용 시 더 나은 경험을 할 수 있습니다.
